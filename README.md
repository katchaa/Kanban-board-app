# Kanban Board

Simple kanban board inspired by Trello, made with Vue 3, Vite and JSON-server.

## Prerequisites

-   [Node.js](https://nodejs.org/en/)

## Project Setup

```sh
npm install
```

## Project launch

#### Vue

Run local dev server

```sh
npm run dev
```

#### JSON-server

Run REST API on localhost connected with JSON file in the root directory.

```sh
npm run server
```
